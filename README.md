# Решение экзаменационного задания "Оптимизация маршрутов" (Вариант 1)

## Условие задачи:

 Логистическая компания столкнулась с проблемой планирования маршрутов для своего автопарка. Маршрут представляет собой путь между городами, который должен удовлетворить нескольким критериям одновременно:

* быть коротким по протяжённости,
* быстрым по времени (например, из-за ограничений в скорости),
* дешёвым по стоимости (например, из-за платных дорог).

Вам необходимо разработать систему, которая по заданной карте дорог и требованиям найдёт не один, а набор маршрутов между двумя городами, каждый из которых является оптимальным по одному из критериев. На финальном этапе система должна выбрать единственный компромиссный маршрут, используя заданные приоритеты.

## Решение задачи

### О проекте:

Решение задания - это программа, которая принимает данные о городах, данные о дорогах между этими городами, запросы на построение маршрутов. Возращает найденные пути по каждому криетрию, а также один самый оптимальный путь по заданным приоритетам. 

Программа состоит из нескольких подпрограмм: 

* программа по обработке входных данных, 
* программа по созданию графа, 
* программа - алгоритм Дейкстры (ищет путь по 1 критерию), 
* программа по объединению найденных путей с помощью алгоритма Дейкстры
* программа по выбору 1 оптимального пути
* программа по форматированию данных для вывода 

### Файловая структура решения:

* **main.py** - исполнительный файл, "движок" программы
* **prepare_data.py** - функция для подготовки входных данных
* **utils.py** - модули кода для работы алгоритма, функции-помощники 
* **formatter.py** - функция для форматирования результа
* **input.py** - входные данные для решения задачи
* **output.py** - ответ 

### Запуск проекта:

1. Склонировать репозиторий локально
2. Добавить входные данные в файл **input.txt**
3. Запустить выполнение программы из корня проекта:
```
python main.py
```
4. Проверить файл **output.txt**.
